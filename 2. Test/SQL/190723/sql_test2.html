=== 문제 미리보기 방지 ===

[문제1]

SCOTT 계정의 테이블을 활용하여 이름(ename)에 F가 들어가는 직원의 근무지(loc)를 검색하라

모범 답안

select loc
from emp e, dept d
where e.deptno = d.deptno
and ename like '%F%';

[문제2]

SCOTT 계정의 테이블을 활용하여 급여(sal)의 등급(grade)가 1인 사람들의 직급(job)을 중복 없이 표현하라

모범 답안

select distinct job
from emp, salgrade
where grade = 1
and sal between losal and hisal;

[문제3]

SCOTT 계정의 테이블을 활용하여 직급(job)이 'SALESMAN'인 사원의 매니저의 고용일(hiredate)를 중복 없이 출력하라

모범 답안

select distinct m.hiredate
from emp s, emp m
where s.job='SALESMAN'
and s.mgr = m.empno;

[문제4]

SCOTT 계정의 테이블을 활용하여 모든 고용인들의 상사 관계를 출력하라

모범 답안

select e.ename 사원, m.ename 상사, m2.ename "상사 상사", m3.ename "상사 상사 상사"
from emp e, emp m, emp m2, emp m3
where e.mgr = m.empno(+) and m.mgr = m2.empno(+) and m2.mgr = m3.empno(+)
order by m2.mgr asc, m.mgr asc, e.mgr asc;

[문제5]

SCOTT 계정의 테이블을 활용하여 가장 낮은 급여를 받는 사람이 속한 부서의 급여의 표준편차를 소수점 첫째자리에서 반올림하여 출력하라. (표준편차 함수 : stddev)

모범 답안

select round(stddev(sal), 0)
from emp, dept
where emp.deptno = dept.deptno
and sal in (select min(sal)
from emp
group by deptno
);
